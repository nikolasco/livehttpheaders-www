Installation

Install LiveHTTPHeaders now

Note: this has only be tested on the following Mozilla version:

  • Mozilla 1.1 on Linux
  • Mozilla 1.2 on Linux and Windows
Don't try it on a lower version of Mozilla!

If you need to remove it, you will need to edit these three files and remove the lines containing 'livehttpheaders' if present:

  • chrome/installed-chrome.txt
  • chrome/overlayinfo/communicator/content/overlays.rdf
  • chrome/overlayinfo/navigator/content/overlays.rdf
