Welcome to the livehttpheaders project.

the goal of this project is to adds information about the HTTP headers in two ways:

  • First by adding a 'Headers' tab in 'View Page Info' of a web page.
  • Second by adding a tool in the 'Tools->Web Development' menu to be able to display http headers in real time (while pages are being downloaded from the Internet.

Note: this has only be tested on Mozilla 1.2.1 on Linux and Windows. Don't try it on a lower version of Mozilla! If you need to remove it, you will need to edit these three files and remove the line containing 'livehttpheaders' if present:

  • chrome/installed-chrome.txt
  • chrome/overlayinfo/communicator/content/overlays.rdf
  • chrome/overlayinfo/navigator/content/overlays.rdf
